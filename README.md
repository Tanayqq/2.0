# MedRef - Healthcare AI Platform

## Vision
MedRef is a production-grade, Retrieval-Augmented Generation (RAG) Clinical Reference Platform. It empowers healthcare professionals by retrieving trusted medical references (openFDA, DailyMed, RxNorm, MedlinePlus, PubMed) and providing precise, strictly cited answers. MedRef is designed with a zero-hallucination tolerance and strictly avoids treatment recommendations or inventing dosages.
